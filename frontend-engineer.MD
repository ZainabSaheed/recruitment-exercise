## Case Study 1

A client runs a small shop in Lagos. He wants a solution to display products online and let customers place orders. It is possible for customers to use their debit card to pay using the solution. 

As a FrontEnd Engineer, 

1.	List the UI features for such a software solution.
2.	Sketch a wireframe for the solution using a pen and plain paper and snap it.
3.	Use React or otherwise to design the Order page where the customer can use a debit card to pay.
4.	When the order button is clicked, use a popover/dialog/modal UI to show the order details and say that it’s successful.
